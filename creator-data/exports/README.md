# creator-data/exports

KOL 名单导入区（不进 Git 的 `.env` 等密钥仍见根目录 `.gitignore`）。

| 目录 | 用途 |
|------|------|
| **`internal/`** | **公司内部** list（领导提供、626 导出、公司口径数据） |
| **`external/`** | **公司外部**补强 list（去重后并入候选池；无合作/从属关系） |

合并前一律对 `blacklists/outreach_do_not_contact.csv` 去重。

Agent 合并时说：`合并 internal list` 或 `合并 external 补强 list`。
