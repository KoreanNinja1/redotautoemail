# RedotPay KOL 外联流程说明

> 给领导看的简版 · 2026-06-25

---

## 一、目前在做什么

**目标：** 找合适的 YouTube 博主合作推广，提高开卡数。

**工具：**

| 工具 | 作用 |
|------|------|
| **KOL Intelligence**（自建网站 API） | 已有 2000+ 频道评分（RAS）、品牌安全、邮箱 |
| **NoxInfluencer** | 补新达人、补邮箱 |
| **本地 CSV** | 主数据、去重、每日待发名单 |
| **Lark** | 半自动话术（英/葡/西） |
| **Gmail** | 实际发信（人工确认后发送） |

---

## 二、目前流程（已跑通）

```text
① 从 KOL Intelligence 拉候选（按国家 + 分数排序）
        ↓
② 去掉 outreach 黑名单（已联系 / 合作中的 1200+ 人）
        ↓
③ 输出 CSV → 导入 Lark → 套话术模板
        ↓
④ BD 过目 → Gmail 发信（名字 + 邮箱）
        ↓
⑤ 发完更新 CSV（已触达 / 是否回复 / follow-up 日期）
```

**原则：**

- **先吃库存**：自建网站里已评分的先发，再用 Nox 找新人
- **每封必审**：不自动发邮件，BD 确认后才发
- **无邮箱也可入库**：缺的用 Nox 或 `/api/jobs/nox-contact` 补，补不到就手动查 YouTube
- **Follow-up**：3 轮，间隔 7 天、14 天

**本周试点：** 沙特市场（7 人待联系，其中 3 人已有邮箱可立即发）

---

## 三、未来流程（可逐步自动化）

```text
① 每日自动：拉候选 → 去重 outreach → API 筛选 → 生成「今日待发 CSV」
        ↓
② AI 生成个性化邮件开头（topic + 频道名）
        ↓
③ Lark 自动填变量 → BD 批量审阅（5–10 封/天起）
        ↓
④ Gmail 发送 + 自动更新状态
        ↓
⑤ 到期自动提醒 follow-up；回复状态回写，AI 排优先级
        ↓
⑥ 库存不足时：Nox 自动搜新 → 过 RAS → 进入队列
```

**可分阶段上：**

| 阶段 | 内容 | 前提 |
|------|------|------|
| **现阶段** | 半自动：脚本出 CSV + Lark 话术 + 人工发信 | ✅ 已具备 |
| **第 2 步** | AI 写个性化开头 + follow-up 提醒名单 | 话术 v2 定稿 |
| **第 3 步** | 每日队列自动生成（按市场/分数） | 主表字段稳定 |
| **第 4 步** | Gmail / Lark 状态回写（减少手工改 CSV） | 跑通 2–4 周人工闭环后 |

**稳态目标：** ~50 封/天（需名单完备、分市场话术、AI 辅助 + BD 审稿）
