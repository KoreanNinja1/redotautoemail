# RedotPay KOL 半自动化外联流程

> Nox 拉人 → RAS 打分 → **≥60 联系 / &lt;60 进黑名单** → Lark 话术 → Gmail 发送

```text
Nox 搜人     →  RAS 打分     →  分流
去 outreach     ≥60 → ready.csv    Lark → Gmail
去 ras&lt;60     &lt;60 → blacklist    （不再搜、不再出 CSV）
```

---

## 1. NoxInfluencer 拉人

BD 确认参数 → 脚本拉名单，**同时排除**：
- `blacklists/outreach_do_not_contact.csv`（已联系）
- `blacklists/ras_below_60.csv`（打过低分）

**输出**：`queues/nox_queue_{市场}_{日期}.csv`（未打分原始名单）

![Nox 搜索](assets/step1-nox-search.png)

---

## 2. 自建网站 RAS 筛选

打分后自动分流：

| RAS | 动作 |
|-----|------|
| **≥ 60** | 写入 `*_ready.csv`，可联系 |
| **&lt; 60 或 Exclude** | 追加 `blacklists/ras_below_60.csv`，以后搜索排除 |

**输出**：
- `*_scored.csv` — 全量打分记录
- `*_ready.csv` — 仅 ≥60，给 Lark 用

![RAS 报告](assets/step2-kol-ras.png)

---

## 3. Lark + Gmail

| 环节 | 谁做 |
|------|------|
| 套话术 | Lark（半自动） |
| 审阅 / 发送 | BD + Gmail（人工） |
| 发完回写 | 追加 `blacklists/outreach_do_not_contact.csv` |

![Lark 文档](assets/step3-lark-doc.png)
