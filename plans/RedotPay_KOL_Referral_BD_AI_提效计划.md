# RedotPay KOL / Referral BD AI 提效计划（审核用）

> 一页纸版本。执行细节见：`BD运营手册-详细版.md`、`每日每周Checklist.md`

---

## 要解决什么

KOL 搜寻、筛选、外联、跟进占用大量时间，影响**开卡获客**。现有工具已具备能力，缺的是串联和每日优先级。

## 怎么做

```text
Nox 找候选 → KOL Intelligence 评分 → AI 排每日优先级 → 当地语言邮件（AI 个性化开头）→ Gmail 触达 / Follow-up
```

- **先做市场**：阿尔及利亚、巴基斯坦、埃及、UAE 等中东/南亚（外币卡需求国）
- **排除**：Unsupported countries（已有清单）
- **粉丝量**：4K–100K
- **不重建评分系统**：复用 KOL Intelligence（已评估 2000+ 频道，日增）

## 已有 vs 待申请

| 已有 | 待申请 |
| --- | --- |
| NoxInfluencer API | KOL Intelligence API（找 **Louis**） |
| KOL Intelligence 网站 | — |
| AI / Codex、Gmail | — |

API 拿到后，每日优先级和个性化邮件可自动化推送给执行人；未拿到前可手工跑通。

## 预期收益

| 维度 | 说明 |
| --- | --- |
| **核心指标** | **开卡数**（老板最关心） |
| 效率 | 减少筛人、排优先级、写邮件时间；稳态目标 ~50 封/天 |
| 成本 | AI token 极低（约 $1–10/月量级），**比扩招 BD 划算** |

## 三阶段

1. **本周**：定框架、主表、话术、向 Louis 申请 API  
2. **下周起**：小批量跑通（5–10 封/天 → 逐步加到 50）  
3. **之后**：每日优先级自动派发 + 数据复盘优化开卡转化  

## 成功标准（2–4 周看）

- 有效触达量提升  
- 正向回复率提升  
- **Referral 开卡数有可追踪提升**  
- BD 每日整理名单时间明显下降  

## 风险与控制

- AI 只做排序和文案草稿，发信前人工确认  
- 不触达 unsupported 国家  
- 刷量 / 刷评论频道不优先  

## 结论

**用现有工具 + AI 提效，低成本提升开卡获客效率，建议批准 MVP 试运行。**
